# Taskly — To-Do List App

A clean, fully featured to-do list application built with vanilla HTML, CSS, and JavaScript. No frameworks, no dependencies — just open the file in any browser and start managing your tasks.

## Features

- **Add tasks** — type and press Enter or click Add
- **Priority levels** — mark tasks as Low, Medium, or High priority
- **Due dates** — set due dates with overdue highlighting
- **Complete & delete** — check off or remove tasks instantly
- **Edit tasks** — update text, priority, due date, and notes via modal
- **Smart views** — filter by All, Active, Completed, Due Today, or High Priority
- **Search** — find tasks instantly by keyword
- **Sort** — sort by newest, due date, priority, or A→Z
- **Progress tracker** — visual progress bar showing completion percentage
- **Clear completed** — bulk-remove finished tasks
- **Toast notifications** — subtle feedback on every action
- **Responsive** — works on desktop and mobile

## Getting Started

### Option 1 — Open directly in browser

1. Download `todo-app.html`
2. Double-click the file to open it in your browser
3. Start adding tasks!

### Option 2 — Clone from GitHub

```bash
git clone https://github.com/YOUR_USERNAME/todo-app.git
cd todo-app
open todo-app.html   # macOS
# or
start todo-app.html  # Windows
```

## How to Use

| Action | How |
|--------|-----|
| Add a task | Type in the input field → press **Enter** or click **Add** |
| Set priority | Click **Low / Medium / High** before adding |
| Set due date | Pick a date in the due date field before adding |
| Complete a task | Click the checkbox on the left |
| Edit a task | Hover over a task → click the ✏️ icon |
| Delete a task | Hover over a task → click the 🗑️ icon |
| Filter tasks | Use the sidebar views or toolbar pills |
| Search | Type in the search bar above the task list |
| Sort tasks | Use the sort dropdown (top right of task list) |
| Clear done | Click **Clear completed** link below the task list |
